
# BSC OPC Agent OS 

关于"偶尔会抽风"这件事，先把话说在前面
商海蟹是送给老粉的免费工具，新粉得有老粉推荐才免费使用额度，（老粉是指公众号关注2个以上的）

我们自己不抓数据、不卖数据、也不靠它赚钱。它能跑起来，全靠站在三家专业数据服务方的肩膀上：

SIF MCP——流量、广告、关键词信号（必选）
Sorftime MCP——市场、1688、TikTok 数据（可选）
卖家精灵 MCP——关键词、竞品、商标（可选）

顺便用大白话讲一下 MCP 是个啥
你可以把 MCP 理解成 ​"AI 和数据之间的一根新数据线"​。以前 AI 想用一家服务商的数据，得让程序员一行一行写代码去对接；现在有了 MCP 这根"标准线"，AI 自己就能直接读数据、用工具。听起来很美好，但这根"线"是 2024 年底才刚刚出现的新东西，全行业都还在摸着石头过河——AI 模型每周在变、MCP 这根线本身的规格也在变、三家数据服务方接口也在跟着升级。
商海蟹做的事，就是把这三根"还在打磨中的线"拢到一起，翻译给 AI 听。所以你可能会偶尔遇到这些情况：

某个模块跑着跑着报错、数据为空、卡在加载中
同一个 ASIN 今天能跑、明天跑不出来
三家数据偶尔对不上号

遇到这些情况，请按这个顺序排查
第一步：先找对应的数据服务商客服。打开右上角设置，看你当前用的是 SIF / Sorftime / 还是卖家精灵，截图把报错信息发给他们家客服，问一句"当前这个 MCP 接口是不是在升级？有没有什么已知问题？​"——他们最清楚自己家服务现在什么状态。
第二步：如果数据服务商那边确认没问题，再来找我们。点击软件右上角"使用反馈"或者公众号后台留言，我们看到就跟。
商海蟹是免费的，但我们对它的态度不是"白送的就将就着用"​——每一次更新都是我们前一晚熬夜跟出来的。只是 AI 和 MCP 都还在快速演进的阶段，希望你能跟我们一起，带着一点耐心，看着它慢慢长大。

## v3.0 (2026-05-17)

**16 个功能模块：**

- **市场分析**（10 个子模块）：供需比 / 品类难度 / 产品筛选 / 竞争扫描 / 关键词机会 / 1688 货源 / 利润测算 / 价格带分析 / 评论洞察 / 成本估算，三数据源并行采集
- **广告分析**（7 个子模块 + 三层下钻）：绩效概览 / 广告活动 / 投放关键词 / 搜索词管理 / 竞价预算 / 广告位优化 / 竞品广告 / 广告架构逐层下钻（ASIN → Campaign → AdGroup）
- **评论洞察**（7 个维度）：痛点分析 / 正面反馈 / 购买因素 / 改进建议 / 高频关键词 / 用户画像 / 使用场景
- **流量下跌诊断**（5 层分析）：总览 / 趋势层 / 结构层 / 关键词层 / 竞争层，逐级定位根因
- **Listing 审核**（9 个维度 42 项检查）：总览 / 标题 / 五点描述 / 图片 / A+ 内容 / 评论 / 关键词覆盖 / 类目准确度 / 定价合理性
- **关键词词库**（6 个子模块）：词库总览 / 需求分析 / 竞争格局 / 特征词分析 / 广告结构 / 广告计划 CSV 导出
- **竞争&出单率分析**（6 个子模块）：态势总览 / 坑位快照 / 出单率分析 / 关键词对抗 / 广告拉升 / 策略计划
- **Alexa for Shopping 建议**：基于 COSMO/Rufus 算法的标题、五点、Q&A 智能优化
- **产品素材生成**：GPT-Image-2 / Gemini 双引擎图生图，10 种图片类型，模板库管理
- **TikTok 竞品分析**（3 个子模块）：相似产品 / 类目查询 / 类目报告，达人带货数据
- **供应链分析**：1688 货源搜索，英文标题智能翻译中文匹配
- **运营工具**：盈亏计算器，20 项输入 14 项输出，FBA 费用自动计算，22 国站点
- **商标查询**（3 个子模块）：全球商标搜索 / 商标详情 / 统计概览
- **补货计划**：基于销量趋势的库存预警与补货建议
- **产品包装建议**：基于产品材质、尺寸的包装方案推荐
- **执行步骤看板**：跨模块数据聚合与待办事项追踪

## 快速开始

1. 解压到任意目录
2. 运行 `BSC-OPC-Agent.exe`
3. 浏览器打开 `http://127.0.0.1:5173`
4. 在设置中配置 API Key
