# BSC OPC Agent OS 

Amazon 产品运营工具：智能图片生成 + Listing 优化。


## v3.0 (2026-05-17)

**16 个功能模块：**

- **市场分析**（10 个子模块）：供需比 / 品类难度 / 产品筛选 / 竞争扫描 / 关键词机会 / 1688 货源 / 利润测算 / 价格带分析 / 评论洞察 / 成本估算，三数据源并行采集
- **广告分析**（7 个子模块 + 三层下钻）：绩效概览 / 广告活动 / 投放关键词 / 搜索词管理 / 竞价预算 / 广告位优化 / 竞品广告 / 广告架构逐层下钻（ASIN → Campaign → AdGroup）
- **评论洞察**（7 个维度）：痛点分析 / 正面反馈 / 购买因素 / 改进建议 / 高频关键词 / 用户画像 / 使用场景
- **流量下跌诊断**（5 层分析）：总览 / 趋势层 / 结构层 / 关键词层 / 竞争层，逐级定位根因
- **Listing 审核**（9 个维度 42 项检查）：总览 / 标题 / 五点描述 / 图片 / A+ 内容 / 评论 / 关键词覆盖 / 类目准确度 / 定价合理性
- **关键词词库**（6 个子模块）：词库总览 / 需求分析 / 竞争格局 / 特征词分析 / 广告结构 / 广告计划 CSV 导出
- **竞争&出单率分析**（6 个子模块）：态势总览 / 坑位快照 / 出单率分析 / 关键词对抗 / 广告拉升 / 策略计划
- **Alexa for Shopping 建议**：基于 COSMO/Rufus 算法的标题、五点、Q&A 智能优化
- **产品素材生成**：GPT-Image-2 / Gemini 双引擎图生图，10 种图片类型，模板库管理
- **TikTok 竞品分析**（3 个子模块）：相似产品 / 类目查询 / 类目报告，达人带货数据
- **供应链分析**：1688 货源搜索，英文标题智能翻译中文匹配
- **运营工具**：盈亏计算器，20 项输入 14 项输出，FBA 费用自动计算，22 国站点
- **商标查询**（3 个子模块）：全球商标搜索 / 商标详情 / 统计概览
- **补货计划**：基于销量趋势的库存预警与补货建议
- **产品包装建议**：基于产品材质、尺寸的包装方案推荐
- **执行步骤看板**：跨模块数据聚合与待办事项追踪

界面字体放大 12.5%，JS 模块化架构，重复查询秒级响应。

## 系统要求

- Windows 10/11 64-bit
- 网络连接 (API 调用)

## 快速开始

1. 解压到任意目录
2. 运行 `BSC-OPC-Agent.exe`
3. 浏览器打开 `http://127.0.0.1:5173`
4. 在设置中配置 API Key
