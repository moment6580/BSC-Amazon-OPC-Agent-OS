## 功能特性

- GPT Image-2 / Gemini 双引擎图生图
- Amazon 产品图智能生成 (主图/A+/品牌故事/高清放大)
- COSMO/Rufus 算法 Listing 优化
- 1688 商品图片采集
- 提示词模板库管理
- SIF MCP 流量与广告分析
- 软件激活与 OTA 在线升级

## 系统要求

- Windows 10/11 64-bit
- 网络连接 (API 调用)

## 快速开始

1. 解压到任意目录
2. 运行 `BSC-OPC-Agent.exe`
3. 浏览器打开 `http://127.0.0.1:5173`
4. 在设置中配置 API Key
