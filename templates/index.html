系统初始化 — 配置指南

首次使用请按以下步骤配置 API 和 MCP 服务，完成后即可正常使用全部功能。
Step 1: 配置 GPT-Image-2（图片生成）必选
OpenAI 兼容的 GPT-Image-2 图生图服务（支持 APIMart、等车等），用于将提示词转换为产品图片。
推荐：注册 APIMart API
也可使用任意 OpenAI 兼容的图生图服务，填入 Base URL + API Key 即可。
Step 2: 注册 SIF MCP（流量分析）必选
亚马逊广告流量分析与关键词运营平台。
注册 SIF MCP
注册后获取 Endpoint 和 API Key，填入设置页面。
Step 3: 注册 卖家精灵 MCP（关键词/竞品）可选
亚马逊关键词研究与竞品分析工具，非必填。
注册 卖家精灵 MCP
注册后获取 API 凭证，填入设置页面。
Step 4: 注册 Sorftime MCP（市场数据）必选
亚马逊市场数据与产品分析平台。
注册 Sorftime MCP
注册后获取 Endpoint 和 API Key，填入设置页面。
Step 5: 配置系统
将以上获取的 API Key 和 Endpoint 填入系统设置，点击测试连接验证。
提示：所有 API Key 仅本地存储，不会上传到任何第三方服务器。
